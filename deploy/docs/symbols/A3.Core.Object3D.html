Class A3.Core.Object3D

Contains properties common to all 3D objects in the engine, like the camera, primitives, the scene [A3.O3D].
Defined in: a3.js.

Class Detail
A3.Core.Object3D()

Author: Paul Lewis.
Field Detail
{A3.Core.Object3D[]} children

The array of the children Object3Ds

{Boolean} dirty

Whether or not this object is dirty. Is automatically set to true if the position, rotation or scale values have been changed. If this is true then all matrices for this object and its children will be recalculated

{A3.Core.Math.Matrix4} matrix

The matrix of object affine transforms such as translation, rotation and scaling. These are in the object space, so around the object's origin.

{A3.Core.Math.Matrix4} matrixWorld

The matrix that we use to position, rotate and scale the object relative to its parent, which ultimately positions the object in world space.

{String} name

The name of the object

{A3.Core.Object3D} parent

The reference to the parent Object3D

{A3.Core.Math.Vector3} position

The position that this object occupies relative to its parent. Ultimately the only object without a parent should be the scene itself. The object matrix's position values will be set from this vector when we update

{A3.Core.Math.Vector3} rotation

The rotation that this object has. The rotations are applied to the object's matrix using Euler XYZ order.

{A3.Core.Math.Vector3} scale

The scale that this object has

{A3.Core.Math.Vector3} target

The target for this object, i.e. what it "looks at" which will be nothing by default, unless it's the basic camera.

{Boolean} visible

The object's visibility.
